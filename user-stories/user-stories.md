# User Stories


## User Registration & Authentication
As a guest, I want to be able to register an account with my email and password, so that I can access personalized features like booking properties and leaving reviews.


## Property Listing Management
As a property owner, I want to be able to create and publish property listings with details (title, description, location, price, amenities, images), so that potential guests can discover and book my properties.


## Property Search & Filtering
As a user, I want to be able to search for properties by location, dates, price range, and amenities, so that I can find accommodations that meet my specific needs and preferences.


## Booking Management
As a registered user, I want to be able to create a booking for a property by selecting check-in/check-out dates and completing payment, so that I can secure my reservation for my trip.


## Payment Processing
As a user making a booking, I want to be able to securely pay for my reservation through integrated payment methods (credit card, PayPal, etc.), so that my booking is confirmed and the property owner receives payment.


## Reviews & Ratings
As a user who has completed a stay, I want to be able to leave a review and rating for the property, so that I can share my experience with other potential guests and help them make informed decisions.


## Notifications System
As a registered user, I want to receive notifications (email/in-app) about booking confirmations, cancellations, and upcoming reservations, so that I stay informed about my bookings and property activities.


## Admin Dashboard
As an administrator, I want to be able to view and manage all users, properties, and bookings through a dashboard, so that I can monitor system activity, resolve issues, and maintain platform integrity.
